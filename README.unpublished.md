## APIs

In addition to raw ZIF byte-served access, server APIs will include:
* REST
* JSON
* Protocol Buffers
* XML-RPC
* MessagePack-RPC
* SOAP
* Thrift
* [IIIF](http://iiif.io)

Serialization formats:
* JSON
* MessagePack
* Gobs (in Go)
* BSON
* Google Protocol Buffers
* Thrift
* HDF5
* XML
* Avro
* Base64/JSON – strings and ArrayBuffers
* don’t forget data can be delivered gzipped…
* HtmlHttpRequest2 supports ArrayBuffers, Blobs, and FormData
